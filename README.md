# ⚡ Elektro-Schaltplan Web-Tool

Willkommen bei deinem interaktiven Tool zum Erstellen von Elektronik-Schaltplänen. Mit dieser Web-App kannst du hoffentlich bald professionelle Schaltpläne direkt im Browser entwerfen – ganz ohne teure Software.

## 🚀 Die Highlights

* Bauteil-Bibliothek:** Nutze fertige Symbole für Elektronik-Bauteile
* Speichern und Laden

## 🛠 Startanleitung

Damit alles reibungslos läuft, beachte bitte den Sicherheitsaspekt moderner Browser (CORS-Policy):

0.  Es ist Python installiert und im Pfad
1.  Lade alle Projektdaten in einen lokalen Ordner.
2.  öffne Terminal (z.B. cmd in Windows)
3.  wechsle in das Verzeichnis von 1. (cd )
4.  gib ein: python -m http.server 8001
4.  Öffne die Adresse `http://127.0.0.1:8001` in deinem Browser.

Alternativ kannst du auch die index.html im Browser öffnen, dann kannst du aber nicht speichern oder neue Dateien laden
